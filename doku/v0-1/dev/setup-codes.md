# tools
## library-manager
### Funktionen:
- secure(^lib^)
  - parameter:
    - lib - lst - die librarys, die installiert werden sollen
  - funktionen:
    - stellt sicher, das eine library installiert ist
    - die library wird in der ^app^ installiert
